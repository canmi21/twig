# Taki

My personal website, and just for fun.

## Commands

```bash
bun install
bun run dev
bun run build
bun run lint
```

## Tech Stack

- TanStack Start
- React SSR
- Vite 8
- Bun
- Tailwind CSS
- shadcn/ui
- lucide-react
- Cloudflare Workers
- Cloudflare D1, R2, and KV
- Drizzle ORM
