# Taki

My personal website, created mainly for fun. A configurable version is planned for the future, but it’s not ready yet. Feel free to explore the source code in the meantime.

## Commands

```bash
bun install
just dev
just build
just lint
```

## Tech Stack

- TanStack Start
- React SSR
- Vite 8
- Bun
- Tailwind CSS
- shadcn/ui
- lucide-react
- Cloudflare Workers
- Cloudflare D1, R2, and KV
- Drizzle ORM
