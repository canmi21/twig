# i18n (Paraglide)

Message sources: `messages/{locale}.json`. Compiled output: `src/lib/paraglide/` (gitignored). Keys use `a.b.c` dot notation as visual namespaces and are accessed via bracket syntax: `m['footer.greeting']({...})`, not `m.footer.greeting()` — dots are part of the flat key, not real nesting.

## Locale roles

| Locale | Role                                                                                                         | Style                                                                                                                                                       |
| ------ | ------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `mw`   | **Design reference.** The locale the author writes the UI in; also the `baseLocale` Paraglide falls back to. | Loose, playful, mixed English + CJK, not-quite-serious.                                                                                                     |
| `en`   | Proper English.                                                                                              | Standard, neutral.                                                                                                                                          |
| `zh`   | Proper Simplified Chinese.                                                                                   | Standard, neutral.                                                                                                                                          |
| `tw`   | Proper Traditional Chinese. **Must use 繁體 characters** — never simplified.                                 | Standard, neutral.                                                                                                                                          |
| `ja`   | Japanese.                                                                                                    | **Cute / casual register** — hiragana-leaning, casual sentence-enders (〜 / よ / ね), avoid stiff formal endings like です／ます where a lighter form fits. |

## Translation rules

- **Not 1:1.** Translate the _meaning_, not the words. Adapt to each language's natural idiom.
- **Length matches `mw`.** The author designs UIs against `mw`, so other locales should aim for similar **rendered width** when the string is user-visible (CJK chars ~2× the width of Latin chars at the same text size — compensate in char count). Aria labels / tooltips / screen-reader-only strings don't count toward the length rule; translate them for correctness and tone only.
- **New message keys must exist in all 5 locale files in the same commit.** Never ship a key that only exists in `mw`.
